How to Launch the Extension
Open the project in Visual Studio Code.
Make sure you have the AL Language extension installed.
Open launch.json and verify:
Server URL is correct (http://localhost)
Server Instance matches your Business Central environment (e.g. BC210)
Authentication is set correctly (Windows / UserPassword)

Press:

F5
Business Central will open in your browser automatically with the extension deployed.
